# Chat Exec

Executes .cfg files through chat commands.

## Usage

Compile the `chat_exec.sp` and install the plugin to any Source Dedicated Server with SourceMod 1.12+ installed.

Then admin players with `ROOT` flag could run `exec` console command through game chat by running `/exec <cfg>`.

The plugin is tested on a Left 4 Dead 2 dedicated server without any issue, other games are not tested but should work as well.
